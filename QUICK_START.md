# Быстрый старт

## Требования

- Java 11 или выше

## Компиляция и запуск

### Компиляция проекта

```bash
javac -d target/classes -encoding UTF-8 -sourcepath src/main/java src/main/java/com/urlshortener/*.java src/main/java/com/urlshortener/model/*.java src/main/java/com/urlshortener/service/*.java src/main/java/com/urlshortener/util/*.java
```

### Запуск приложения

```bash
java -cp target/classes com.urlshortener.UrlShortenerApp
```

### Или используйте скрипт (Windows)

```bash
compile_and_run.bat
```

## Первые шаги

1. **Запустите приложение** (см. выше)

2. **Создайте первую ссылку:**
   - Выберите пункт меню `1`
   - Введите URL (например: `https://www.baeldung.com/java-9-http-client`)
   - Введите лимит переходов (или нажмите Enter для 10)
   - Введите время жизни ссылки (например: `24ч`, `3д`, `12 часов` или Enter для 24 часов)
   - **Сохраните ваш User ID!**

3. **Перейдите по ссылке:**
   - Выберите пункт меню `2`
   - Введите короткую ссылку
   - Браузер откроется автоматически

## Основные команды меню

- `1` - Создать короткую ссылку
- `2` - Перейти по короткой ссылке
- `3` - Просмотреть мои ссылки
- `4` - Редактировать ссылку (изменить лимит переходов и время жизни)
- `5` - Удалить ссылку
- `6` - Показать мой User ID
- `7` - Установить User ID
- `0` - Выход

## Важные замечания

⚠️ **Сохраните ваш User ID!** Он нужен для доступа к вашим ссылкам.

✅ **Данные сохраняются на диск** - все ссылки и пользователи автоматически сохраняются в директорию `data/` и загружаются при следующем запуске.

⚠️ **Время жизни ссылки** - задаётся пользователем при создании (по умолчанию 24 часа). Просроченные ссылки автоматически удаляются.
